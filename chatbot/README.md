DecodeLabs Internship - Rule-Based Chatbot

A modern, user-friendly desktop chatbot built with Python and Tkinter as part of the DecodeLabs Internship Program.


##  Features

- Modern Dark/Light Theme with smooth toggle
- Real-time Chat Interface with message bubbles
- Chat History automatically saved using SQLite
- Typing Animation for realistic AI feel
- Multiple Commands (`/clear`, `/help`, `/exit`)
- Keyboard Support (Press `Enter` to send)
- Responsive & Clean UI

## Technologies Used

- Python 3
- Tkinter(for GUI)
- SQLite3(for chat history)
- CustomTkinter styling
